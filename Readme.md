# Detector de Morango 🍓

Projeto simples que identifica se um morango é **bom** ou **ruim** usando um modelo treinado no **Teachable Machine**.

---

## Tecnologias

- HTML, CSS e JavaScript
- TensorFlow.js
- Teachable Machine (modelo de classificação)
- GitHub Pages (para rodar online)

---

## Como usar

1. Abra o site: [Detector de Morango](https://yfolli.github.io/Detector-morango/)  
2. Clique em **Escolher arquivo** e selecione uma imagem de morango.  
3. O resultado aparece na tela com a classificação **Bom** ou **Ruim** e a porcentagem de confiança.

---

## Sobre o modelo

- Classes: `Bom` e `Ruim`  
- Treinado com imagens de morangos coletadas do Kaggle  
- Exportado para rodar direto no navegador com TensorFlow.js
